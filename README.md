Citizens scan a QR code on the bin or upload a photo with GPS location.

The report is stored in a centralized database.

Municipal administrators verify the issue via the dashboard.

Tasks are assigned to field workers.

Once resolved, the status is updated with proof images.

Users receive eco points for verified reports.

On reaching milestones, users earn certificates and tree saplings.
